---------------------------------
READ ME
---------------------------------
In order to use the program, Apache SOLR must be downloaded.

Enter the root directory of SOLR in the Linux termnal and submit the following comands:
$  bin/solr start -c 8900
$  bin/solr create_core -c solrservices -p 8900

After the commands are entered and are working correctly, the app will be ready to use.
